# User System Project
A command-line management system built with Python.

## Features
- User registration
- User login
- Admin role system
- Password hashing
- Input validation
- Account lock system
- Admin unlock user
- JSON data persistence
- Modular project structure

## Tech Stack
- Python
- JSON
- hashlib
- Git
- GitHub

## Project Structure
```text
user_system_project/
|----main.py
|----auth.py
|----validators.py
|----security.py
|----data.py
|----users.json
|----README.md
```

# How to Run
```bash
pyhon main.py
```
